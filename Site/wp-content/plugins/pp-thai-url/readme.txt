ช่วยให้เว็บของคุณมี URL ภาษาไทยที่ยาวขึ้น
== Description ==
เว็บไซต์ที่ทำด้วย Wordpress นั้นเป็นธรรมดาที่ URL จะถูกตัดให้เหลือแค่ 200 ตัวอักษร เนื่องจาก Browser เข้าใจเฉพาะภาษาอังกฤษจึงไม่สามารถเข้าใจภาษาไทยได้ จึงจำเป็นที่จะต้องเข้ารหัส URL ที่ไม่ใช่ภาษาอังกฤษเพื่อแปลงให้เป็นภาษาอังกฤษซะ เช่น 1 ตัวอักษรในภาษาไทยจะถูกแปลงเป็นภาษาอังกฤษ ถึง 3 ตัวอักษร เท่ากับว่า คนไทยสามารถพิพ์ได้แค่ 66 ตัวอักษรเท่านั้น  ทำให้บางที URL สั้นกว่าที่คิด.

=  ปลั๊กอินนี้จะช่วยอะไรคุณบ้าง:  =

*   ช่วยให้สามารถใส่ Slug หรือชื่อ Post Page ได้ถึง 2000 ตัวอักษร
*   ช่วยให้ Wordpress ตัดคำใน URL ที่เป็นภาษาไทยได้ยาวขึ้นไม่กั๊กแล้ว (ความจริงใช้ได้ทุกภาษาเลยหละ)
*   สามารถเพิ่ม แก้ไข ลบ Slug หรือ URLได้ตามปกติ 
*   เมื่อติดตั้งชื่อ URL ทั้งหมดจะถูกแก้ไขให้โดยอัตโนมัติ
*   ทดสอบแล้วใช้ได้ทั้งเว็บ Wordpress ธรรมดา และ Wordpress Mu.
*   เมื่อทำแล้วผมคิดว่า SEO ของคุณน่าจะดีขึ้นมาทันตาเห็น.
*  ไม่ต้องห่วงเมื่อคุณถอดการติดตั้ง ระบบจะคืนค่าทั้งหมดกลับเป็นเหมือนเดิมโดยอัตโนมัติ รวมทั้งแก้ไข URL ทุก Post และ Page จากยาวๆ ให้สั้นลงเอง Link จะได้ไม่เสีย

== Frequently Asked Questions ==

= ใช้แล้วเว็บจะพังหรือไม่? =

ผมแนะนำว่าให้คุณลองใช้กับเว็บที่ไม่ใช่เว็บจริงของคุณก่อน ถ้าใช้ได้ดีมั่นใจในปลั๊กอินตัวนี้แล้ว ก็ค่อยใช้กับเว็บจริงๆของคุณ.

= ใช้ปลั๊กอินตัวนี้แล้วมีผลอย่างไรกับ SEO บ้าง? =

ผมขอตอบแบบง่ายๆ ครับ ถ้าเว็บคุณทำ SEO เอาไว้ก่อนแล้วด้วย Link เดิม แน่นอนเมื่อติดตั้งปลั๊กอินนี้ไปแล้ว Link เดิมจะใช้ไม่ได้นะ เพราะ URL ทั้งหมดจะถูกเปลี่ยนใหม่เลย อาจทำให้คนที่เข้ามาดูเว็บคุณรู้สึก งุนงงๆ ได้ แนะนำว่า ควรใช้ปลั๊กอินนี้ก่อนการปั่น SEO ของคุณครับ เวลาคนเข้ามาที่เว็บคุณจากเว็บอื่นๆ ที่คุณทำลิ้งเข้ามาจะได้ไม่มีปัญหาในภายหลังครับ

= แล้ว URL ที่ Google เคย Index ไปแล้วหละ ยังเป็น URL เดิมอยู่เลยทั้งๆ ที่เปลี่ยนแล้วนะ =

ไม่ต้องตกใจครับ เฮียกูเกิ้ล เค้าบางทีกว่าจะเปลี่ยนให้ อาจจะนานเกิน 1 อาทิตย์ครับ แต่ผมแนะนำว่า ให้เขียนบทความใหม่ซักหนึ่งบทความเมื่อคุณกด Publish ระบบ Update Service จะทำงานทันที เค้าจะไปเรียกพี่กูเกิลมาเก็บข้อมูลใหม่ครับ น่าจะเร็วขึ้นนิดนึง  พอเฮียเข้ามาเก็บบทความที่เคย Index ไว้จะถูกปรับให้อัตโนมัติครับ ฉะนั้นมั่นใจได้หายห่วง ลิ้งจาก Google ไม่เสียแน่นอน แต่แรกๆทำใจไว้ก่อนเลย คนอาจจะหายไปซักระยะ แต่พอเปลี่ยนแล้วคนจะกลับเข้ามามากขึ้นเหมือนเดิม หรือมากกว่าเดิมอีก แน่นอนครับ ฟันธง!
